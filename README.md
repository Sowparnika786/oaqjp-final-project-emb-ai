# Emotion Detection Application using Watson NLP

## Project Overview

This project is an emotion detection application that uses IBM Watson NLP to analyze text and identify emotions such as anger, disgust, fear, joy, and sadness.

## Features

- Detects emotions from text input
- Uses IBM Watson NLP Emotion Prediction service
- Returns emotion scores for anger, disgust, fear, joy, and sadness

## Technologies Used

- Python
- Requests Library
- IBM Watson NLP

## Project Structure

- `emotion_detection.py` - Contains the emotion detection function
- `README.md` - Project documentation
